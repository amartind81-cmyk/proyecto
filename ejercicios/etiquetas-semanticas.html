¿Qué es DHCP?


Definición de DHCP DHCP (Dynamic Host Configuration Protocol) es un protocolo de red que permite a los dispositivos obtener automáticamente una dirección IP y otros parámetros de configuración necesarios para comunicarse en una red. Fue desarrollado para simplificar la administración de direcciones IP en redes grandes, evitando la necesidad de asignar manualmente direcciones a cada dispositivo.


Como actúa DHCP Cuando un dispositivo se conecta a una red que utiliza DHCP, envía una solicitud de configuración (conocida como DHCPDISCOVER) para obtener una dirección IP. Un servidor DHCP en la red responde con una oferta (DHCPOFFER) que incluye una dirección IP disponible y otros parámetros de configuración, como la máscara de subred, la puerta de enlace predeterminada y los servidores DNS. El dispositivo acepta la oferta enviando un mensaje de solicitud (DHCPREQUEST), y el servidor confirma la asignación con un mensaje de confirmación (DHCPACK). La dirección IP asignada tiene un tiempo limitado (conocido como "lease"), después del cual el dispositivo debe renovar la asignación o solicitar una nueva dirección.
